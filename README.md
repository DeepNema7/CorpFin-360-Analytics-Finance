# CorpFin-360 — AI-Powered Financial Analytics Dashboard

## 📊 Overview
Built an interactive financial analytics dashboard using Excel and Power BI to simulate AI-driven business insights.

## 🚀 Features
- Revenue, Cost & Profit Analysis
- Regional Performance Tracking
- Interactive Filters (Region, Month, Company)
- KPI-driven insights

## 🛠 Tools Used
- Excel
- Power BI
